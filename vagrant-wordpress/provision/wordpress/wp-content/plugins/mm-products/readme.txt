Mile Marker Products enables a "Product" custom post type in WordPress. 


== Description ==

When you install this plugin, "Products" will appear in addition to "Pages"
and "Posts". You can add any valid product from Shopatron and the Product will
fetch details about the Product from Shopatron.


== Installation ==

Copy the plugin folder into `wp-content/plugins`.
